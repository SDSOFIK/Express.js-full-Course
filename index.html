Web API & Server

Express.js দিয়ে মিনিটের মধ্যে REST API, routing, middleware — সব কিছু তৈরি করো।

// প্রথম Express server
const express = require('express');
const app = express();

app.get('/', (req, res) => {
  res.send('হ্যালো World!');
});

app.listen(3000);
// → Server চলছে port 3000-এ

কেন Express.js?

সহজ, দ্রুত, এবং শক্তিশালী — প্রতিটি project-এর জন্য

🛣️

Routing

সহজে যেকোনো URL route define করো এবং dynamic parameter ব্যবহার করো।

🔧

Middleware

Request ও response এর মাঝে custom logic চালাও — auth, logging, validation।

🔌

REST API

GET, POST, PUT, DELETE সব HTTP method সাপোর্ট করে অনায়াসে।

🚀

দ্রুত গতি

Lightweight হওয়ায় production-এ অনেক দ্রুত কাজ করে কম resource-এ।

কিভাবে শুরু করবে?

মাত্র ৩টি ধাপে Express.js সেটআপ করো

১

Install করো

Terminal-এ npm install express চালাও।

২

Server তৈরি করো

একটা app.js file বানাও এবং উপরের code লেখো।

৩

Run করো

node app.js চালাও এবং browser-এ localhost:3000 খোলো।

বানানো হয়েছে Express.js দিয়ে — ভালোবাসা সহ ❤️
